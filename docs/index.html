Uproar Linux










Welcome to Uproar

An arch based project designed for low end PCs.












What is Uproar

Uproar is a project created for low end hardware that can't function normally with the pre-installed operating system. Examples being lower end Chromebooks, old pcs, and most Windows tablets. The project is a script that you run in the Arch Linux live cd/usb, and it will setup the Uproar project on your computer.
